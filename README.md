# 🚗 Uber Analysis Dashboard

## 📊 Overview
This project analyzes Uber ride data to understand trip trends, revenue, and demand patterns.

## 🛠 Tools Used
- Power BI / Excel
- SQL (optional)
- GitHub

## 📈 Key Insights
- Peak demand during office hours
- Higher revenue on weekends
- Card payments dominate

## 📂 Files
- uber.xlsx → Raw dataset
- uber_dashboard.pbix → Dashboard file
